# Описание функций проекта

## circle.py

### area(r)
#### Описание
Принимает число r - радиус окружности, возвращает квадрат числа r, умноженный на константу math.pi - площадь окружности с радиусом r. Использует константу библиотеки math math.pi.
#### Пример работы
area(1) -> 3.141592653589793
area(2) -> 6.283185307179586

### perimeter(r)
#### Описание
Принимает число r - радиус окружности, возвращает произведение 2, константы math.pi и r - длину окружности с радиусом r. Использует константу библиотеки math math.pi.
#### Пример работы
perimeter(4) -> 25.132741228718345
perimeter(2) -> 12.566370614359172


## square.py

### area(a)
#### Описание
Принимает число a - длину стороны квадрата, возвращает квадрат a - площадь квадрата.
#### Пример работы
area(2) -> 4
area(3) -> 9

### perimeter(a)
#### Описание
Принимает число a - длину стороны квадрата, возвращает число a, умноженное на 4 - периметр квадрата.
#### Пример работы
perimeter(4) -> 16
perimeter(2) -> 8


## triangle.py

### area(a, h)
#### Описание
Принимает числа a - длину стороны треугольника, h - длину высоты треугольника, проведенной к этой стороне, возвращает произведение a и h, делённое на 2 - площадь треугольника со стороной a и высотой h.
#### Пример работы
area(2, 2) -> 2
area(10, 2) -> 10

### perimeter(a, b, c)
#### Описание
Принимает числа a, b, c - длины сторон треугольника, возвращает их сумму - периметр треугольника.
#### Пример работы
perimeter(1, 2, 3) -> 6
perimeter(2, 2, 8) -> 12

## rectangle.py

### area(a, b)
#### Описание
Принимает числа a, b - длины сторон прямоугольника, которые не являются параллельными по отношению друг к другу, возвращает их произведение - площадь прямоугольника.
#### Пример работы
area(2, 3) -> 6
area(4, 7) -> 28

### perimeter(a, b)
#### Описание
Принимает числа a, b - длины сторон прямоугольника, не являющиеся параллельными по отношению друг к другу, возвращает их сумму, умноженную на 2 - периметр прямоугольника.
#### Пример работы
perimeter(4, 4) -> 16
perimeter(2, 3) -> 10
