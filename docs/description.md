# Общее описание проекта
Данный проект позволяет вычислять площадь и периметр 4 фигур: окружности, треугольника, квадрата и прямоуголька.

## Используемые библиотеки
math

## Реализованные формулы площади
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = a * h / 2

## Реализованные формулы периметра
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a + b + c
